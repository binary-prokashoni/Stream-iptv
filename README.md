# Stream · IPTV Player

A sleek, free live TV client for streaming M3U playlists — no subscription, no cost.

🌐 **[stream-free-iptv.vercel.app](https://stream-free-iptv.vercel.app)**

---

## Features

- 10,000+ international live TV channels out of the box
- Add & manage your own M3U playlists
- Filter channels by country
- Favorites saved locally on your device
- Sleep timer for auto-stop playback
- Video fit toggle (Contain / Cover)
- Works entirely in the browser — no install required

---

## Usage

1. Open the [web app](https://stream-free-iptv.vercel.app)
2. A curated playlist loads automatically — start watching right away
3. To add your own playlist → tap **☰ → Playlist Settings → Add New Playlist**
4. Switch between playlists anytime using the playlist button

## FAQ

**Stream not loading?**  
Check your internet connection. Some channels may be geo-restricted or temporarily offline.

**Video looks stretched?**  
Use the **Fit** button in the player controls to toggle between Contain and Cover modes.
